> 使用说明（简要说明使用操作方法）：
- 1.运行环境
1)Tomcat6以上
2)JDK1.7
3)Eclipse
4)Chrome浏览器
5)Mysql5.5
6)Window10
- 2.安装步骤
 - 1)把项目导入到Eclipse
点击File>>import>>Existing Projects into Workspace>>选择源代码Forum
中文环境下点击文件>>导入>>现有项目到工作空间中>>选择源代码Forum
